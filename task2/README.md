# README

## Overview

This Jupyter Notebook analyzes experimental data from the dataset `Antisocial punishment across societies`. It utilizes Pandas for data processing and Plotly for visualization. The analysis likely focuses on group behavior, economic decision-making, or social experiments.

## Dataset

- **Source**: [Dryad Repository](https://datadryad.org/dataset/doi%3A10.5061/dryad.87301)
- **File**: `data/HerrmannThoeniGaechterDATA.csv`
- **Structure**: Contains data on economic or social behavior experiments.

## Dependencies

To run the notebook, install the following Python packages:

```bash
pip install pandas plotly
```

## Usage

1. Place the dataset in the `data/` directory.
2. Open the Jupyter Notebook:
   ```bash
   jupyter notebook mmde-task2.ipynb
   ```
3. Run the cells sequentially to process and visualize the data.

## Output

- The notebook calculates average contributions by different groups.
- It generates visualizations using Plotly.
